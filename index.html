🔥 One-time Setup

LiveBoard needs a free Firebase project to sync drawings in real time.
Takes about 3 minutes — only once!

1
Create a free Firebase project console.firebase.google.com → Sign in with Google → Add project → give it any name → Continue
2
Enable Realtime Database Left sidebar → Build → Realtime Database → Create database → Start in test mode → Enable
3
Get your config ⚙️ Project settings → Your apps → </> Web → Register app → copy the firebaseConfig object
4
Paste it below and save Paste the whole JSON object (the { … } part)

Config is saved in your browser only. Your visitors don't need to do this setup.
