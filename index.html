One-time Setup

LiveBoard needs a free Firebase project to sync drawings in real time.
This takes about 3 minutes — only needed once!

1
Create a free Firebase project console.firebase.google.com → Sign in with Google → Add project → give it a name → Continue
2
Enable Realtime Database Build → Realtime Database → Create database → Start in test mode → Enable
3
Get your config Project Settings (⚙️) → Your apps → Web (</>) → Register app → copy the firebaseConfig object
4
Paste it below Paste the whole config JSON (the part inside the curly braces { })

Config is saved in your browser. Your friends who join your board links won't need to do this — only you (the host) needs Firebase set up on Vercel env vars for production.
